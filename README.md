Manutenção de vida saúdavel.

Este projeto foi designado para concorrer na feira do conhecimento municipal, realizada em Chapecó/SC.
(2017)

Colaboração:

Materiais usados:

 - Arduino 101;
 - Sensor de temperatura;
 - Jump wires;
 - LCD 16x2;
 - Nan. Protoboard;
 - Bateria (9v);
 - Molde impresso.
